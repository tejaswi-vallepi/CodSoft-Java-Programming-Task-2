# Student Grade Calculator

## Overview

The Student Grade Calculator is a Python-based console application developed to calculate a student's total marks, average percentage, and final grade based on marks entered for multiple subjects. The program includes input validation, grade classification, and performance feedback, making it a simple and effective academic utility.

## Features

* Accepts marks for multiple subjects.
* Calculates total marks obtained.
* Computes average percentage automatically.
* Assigns grades based on percentage.
* Provides performance feedback messages.
* Validates user input to prevent invalid entries.
* Handles exceptions for non-numeric input.
* User-friendly console interface.

## Technologies Used

* Python 3
* Functions
* Loops
* Conditional Statements
* Exception Handling
* User Input Validation

## Grade Criteria

| Percentage | Grade |
| ---------- | ----- |
| 90 - 100   | A+    |
| 80 - 89    | A     |
| 70 - 79    | B     |
| 60 - 69    | C     |
| 50 - 59    | D     |
| Below 50   | F     |

## How the Program Works

1. The user enters the number of subjects.
2. Marks for each subject are entered individually.
3. The program validates that marks are between 0 and 100.
4. Total marks and average percentage are calculated.
5. A grade is assigned according to the grading criteria.
6. The final result summary is displayed along with a performance message.
